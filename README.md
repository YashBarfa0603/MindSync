# 🧠 MindSync – AI-Powered GitHub Profile Analyzer

## 🚀 Overview

MindSync is an AI-powered platform that analyzes a developer’s GitHub profile to extract skills, evaluate projects, and match candidates with suitable job roles.
It simplifies the hiring process by converting unstructured GitHub data into structured insights using AI.

---

## 💡 Problem Statement

Recruiters often struggle to evaluate candidates based only on resumes.
GitHub profiles contain valuable information, but analyzing them manually is time-consuming and inconsistent.

---

## ✅ Solution

MindSync automates GitHub profile analysis using AI to:

* Extract technical skills
* Analyze repositories and contributions
* Match candidates with relevant job roles

---

## 🔥 Features

* 🔍 GitHub Profile Analyzer
* 🤖 AI-Based Skill Extraction (LLM Powered)
* 📊 Project & Repository Evaluation
* 💼 Job Role Matching System
* ✏️ Editable Job Description & Skills
* 📄 Optional Resume Upload
* ⚡ Fast and User-Friendly Interface

---

## 🛠️ Tech Stack

* **Frontend:** Flutter / Streamlit
* **Backend:** Python
* **AI/ML:** LLM (OpenAI / HuggingFace)
* **APIs:** GitHub API


---

## ⚙️ How It Works

1. User enters GitHub profile link
2. System fetches repositories using GitHub API
3. AI extracts skills and technologies
4. Projects are analyzed for quality and relevance
5. Candidate is matched with job roles
6. Results are displayed in a structured dashboard

## 🧪 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/mindsync.git

# Navigate to project folder
cd mindsync

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
